# RM-Web-Page
RM的web页面

This single page app allows user to control the Radiomaster's T8L, including:
- Map and configure channels
- Calibrate
- Firmware Update for T8L and it's ELRS
- Change modes 
- View and change trims
- Configure ELRS
- Communicate via UART
- View devices status


Currently supported languages:
- English
- 中文 (Chinese)
- Українська (Ukrainian)
